# ZeroxAI VPN — Desktop & Mobile Client

## 📦 Состав

| Файл | Платформа | Описание |
|------|-----------|----------|
| `dist/ZeroxAI_VPN.exe` | Windows | Готовый .exe (12 МБ) |
| `ZeroxAI_VPN.bat` | Windows | Ярлык для запуска |
| `zeroxai_vpn.py` | — | Исходный код (Python) |
| `mobile/index.html` | Android / iOS | Установить как PWA |
| `mobile/manifest.json` | Android / iOS | Манифест PWA |

## 🪟 Windows (.exe)

**Как использовать:**
1. Открой `dist/ZeroxAI_VPN.exe`
2. Нажми **+ Добавить** или перетащи `.conf` файл
3. Выбери конфиг из списка → **Подключиться**
4. WireGuard сам откроется с этим конфигом

**Где взять .conf файл:** Открой Telegram бот → 🌍 VPN → выбери сервер → 🔌 Подключиться → 📎 Скачать .conf

## 📱 Телефон (Android / iOS) — PWA

**Как установить:**
1. Открой `mobile/index.html` в браузере на телефоне
2. Нажми **«Поделиться»** → **«На экран «Домой»** (Safari)
   или **«Установить приложение»** (Chrome)
3. Откроется как родное приложение

**Как подключиться:**
1. Нажми **📂** — выбери скачанный из бота `.conf` файл
2. Нажми **Подключиться** — конфиг скачается, появится QR-код
3. Открой WireGuard → сканируй QR → готово
